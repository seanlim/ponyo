Ponyo is a 2D C++ game engine built on DirectX.

# Requirements

- Visual Studio (Windows)
- DirectX 9

# Manual

Main engine code resides in `core/`.

## Default game commands

`SHIFT` + `Q` to quit

`F2` to toggle FPS

`F3` to toggle debugger

## BREAKOUT!

Driver game to test engine in `src/`
